# microgen Documentation

The maintained generator is `github.com/dreamsxin/go-kit/v2/cmd/microgen`.

Use:

- [v2/MICROGEN.md](v2/MICROGEN.md) for CLI usage and generated ownership.
- [v2/ARCHITECTURE.md](v2/ARCHITECTURE.md) for IR and package boundaries.
- [v2/ROADMAP.md](v2/ROADMAP.md) for future implementation scope.
- [v2/MAINTAINING.md](v2/MAINTAINING.md) for generator validation.
- [v2/MIGRATION.md](v2/MIGRATION.md) for migration from the v1 generator.

Root-level `MICROGEN_*_DESIGN.md` and `MICROGEN_COMPATIBILITY.md` files describe
the legacy v1 generator and are retained only for v1 maintenance history.
